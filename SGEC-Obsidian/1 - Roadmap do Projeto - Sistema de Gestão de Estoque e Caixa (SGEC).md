# 🔹 V1.0 — Fundação do Projeto

## Objetivo

Criar a estrutura do sistema e desenvolver toda a lógica do estoque.

Ainda não existe banco de dados.

Ainda não existe interface.

Tudo acontece dentro do Python.

---

### Funcionalidades

- Cadastro de produtos
- Editar produtos
- Remover produtos
- Entrada no estoque
- Saída do estoque
- Listagem do estoque

---

### Assuntos estudados

- Organização de projetos Python
- Programação Orientada a Objetos
- Classes
- Objetos
- Métodos
- Encapsulamento
- Modelagem de domínio
- Responsabilidade das classes
- Estruturas de dados
- Git
- Commits

---

### Objetivo de aprendizagem

Aprender a construir um sistema organizado.
